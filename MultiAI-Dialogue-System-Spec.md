# MultiAI 对话系统规格说明书

## 1. 项目概述
本软件是一个开源、本地优先的多大模型协同对话系统，核心目标是：让人类通过一个「牵引主题」，驱动多个 AI 模型按顺序/并行思考、接力输出，并将全部过程自动沉淀为结构化 Markdown 日志。

### 核心价值
- ✅ **主题聚焦**：所有 AI 输出围绕同一主题滚动深化，避免发散
- ✅ **模型平权**：OpenAI / Claude / Ollama / Qwen 等可自由混搭，各司其职
- ✅ **过程可溯**：每轮输出、每次追问、每个总结均时间戳+角色标记，永久存档
- ✅ **人类掌控**：随时暂停、插入追问、终止并要求总结，全程主导权在你

## 2. 系统架构
- 前端HTML
- 后端Python
- 存储：本地文件系统（`.md` 日志 + `messages.db` 元数据）

## 3. 功能模块设计
### 3.1 对话页面（主窗口）
- 左侧：对话会话列表（支持新建/重命名/删除）
- 右侧：当前会话区域，含：
  - 🎯 牵引主题输入框（必填，支持中文/英文/emoji）
  - ⚙️ 配置面板（选择自动停止条件：循环次数/时长/Token上限、默认Prompt模板）
  - 💬 实时对话区（带滚动、语法高亮、复制按钮）
  - ▶️ 启动 / ⏸️ 暂停 / ❌ 终止 按钮
  - ➕ 追问输入框（发送后自动以 `用户:` 开头写入日志）

### 3.2 全局配置页面（设置窗口）
- 表格形式管理模型：
  | 别名 | 模型名称 | API 类型 | Base URL | API Key（掩码） | 默认Prompt | 启用 |
  |------|----------|----------|----------|----------------|------------|------|
  | 智言 | gpt-4o   | OpenAI   | https://api.openai.com/v1 | •••••• | "你是一位严谨的..." | ✅ |
- 支持：添加/编辑/测试连接/设为默认/导出为 JSON

## 4. 核心流程：牵引式多模型循环
1. 用户输入「牵引主题」（例：`如何用烧腊外卖提升社区老年用户复购率？`）
2. 系统按配置顺序调用状态为启用的模型 A → B → C...
3. 每次请求携带：
   - 当前牵引内容
   - 历史全部输出（从该主题 `.md` 文件中读取）
   - 当前模型专属 Prompt
4. 输出实时追加到 `./data/topics/{主题摘要}/topic.md`，格式：
   ```md
   ### [2025-01-17 15:30:22] 智言（gpt-4o）
   > 基于您提供的烧腊外卖现状，我建议从三方面切入：
   > 1. **时段优化**：早市（7–9点）推出「孝心早餐套餐」...
   ```
5. 循环条件满足即停（如：总 Token ≥ 8192，或运行 ≥ 10 分钟，或完成 5 轮）
6. 用户可随时点击「追问」，输入：`能否给出具体话术示例？` → 自动写入 `.md` 并触发下一轮
7. 点击「终止讨论」→ 自动追加：`### [2025-01-17 15:45:01] 系统
> 本次讨论已终止。请基于以上全部内容，做出结构化总结。` → 最终模型输出总结段落

## 5. 配置存储设计
- `./data/config/models.json`：模型配置（JSON Schema 严格校验）
- `./data/config/settings.json`：全局设置（默认别名规则、日志路径等）
- `./data/topics/{主题摘要}/topic.md`：纯文本 Markdown 日志（人类可读、Git 友好）
- `./data/topics/{主题摘要}/messages.db`：SQLite（记录时间、模型、Token 数、状态）

## 6. 界面交互细节
- 快捷键：`Ctrl+N` 新建会话｜`Ctrl+R` 重启循环｜`Ctrl+Q` 追问｜`Esc` 暂停
- 状态栏显示：`🟢 运行中｜🔄 第3轮｜⏱️ 2m14s｜🔢 1,247 tokens`
- 错误弹窗：API Key 无效 / 网络超时 / 模型返回空，均明确提示

## 7. 扩展性设计
- 导出功能：一键生成 PDF / HTML / Word 报告（含图表）

## 8. 非功能性需求
- 性能：单轮响应 < 3s（本地 Ollama 模型）｜< 8s（远程 API）
- 安全：API Key 永不上传、不记录、不日志；内存中明文仅存活 5 秒
- 兼容：Windows 10/11（x64），Python 3.11+，无需管理员权限
